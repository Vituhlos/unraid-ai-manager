# Unraid AI Manager plugin

This plugin installs and manages the Unraid-side helper daemon.

The PC-side MCP server is not installed here. It runs on the user's PC and connects to this helper over HTTP, preferably through an SSH tunnel.

## Czech

Tento plugin instaluje a spravuje helper daemon běžící na Unraidu.

MCP server pro PC se tímto pluginem neinstaluje. Běží na uživatelově PC a k helperu se připojuje přes HTTP, ideálně přes SSH tunnel.
